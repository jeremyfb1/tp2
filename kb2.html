Explications

:hover

La pseudo-classe :hover est une fonction qu'on applique sur un élément comme un lien ou un div et ca fait que quand on passe la souris au dessus de l'élément, il a les fonctionalités CSS qu'on lui a mit

Exemple

Opacité d'une image

"opacity" est une propriété CSS qui permet de gérer l'opacité d'une image. On la déclare comme ci : opacity: "nombre allant de 0 à 1"

Par exemple, ceci est une image avec une opacité de 0.5

background-image: linear-gradient()

background-image: linear-gradient() est une propriété CSS qu'on applique a l'arrière plan d'un élément.

Dans les paranthèses, on commence par déclaré la direction du dégradé (par exemple : to left) ensuite, les paramètres suivants sont les couleurs qui font partis du dégradé (par exemple : pink, purple)

Dans l'exemple derrière cette section la déclaration du dégradé est : background-image: linear-gradient(to left, pink, purple)
